# Archivum ‚Äî Secure Evidence Capture

Archivum is a privacy-first web application that helps individuals safely collect, organize, and document evidence of harassment, abuse, or misconduct. It enables users to compile screenshots, links, messages, and a chronological timeline into a structured incident report generated with AI assistance.

---

## üß© Project Description

Many people struggle to document harassment or abuse in a clear, organized, and credible way. Evidence is often scattered across chats, screenshots, and links, making it difficult to present a coherent account of events.

Archivum solves this problem by providing a simple, secure interface where users can collect all forms of evidence in one place and generate a factual, structured incident documentation PDF.

**Technologies used:**
- HTML5
- CSS3
- Vanilla JavaScript
- jsPDF (PDF generation)
- Google Gemini API (AI-generated documentation)

All processing happens client-side to maximize privacy and user control.

---

## üéØ Purpose

The motivation behind Archivum is to reduce the emotional and technical burden of documenting difficult experiences. During stressful situations, organizing evidence should not be an additional obstacle.

Archivum matters because it:
- Encourages responsible documentation
- Protects user privacy by avoiding permanent storage
- Helps users present events clearly and chronologically

If developed further, Archivum could support survivors, students, employees, journalists, or activists by making evidence collection safer, clearer, and more accessible.

## ‚öôÔ∏è How It Works

Users can:
- Upload screenshots or media files
- Add links to online posts or profiles
- Paste chat messages or text evidence
- Build a chronological timeline of events
- Generate a structured incident documentation PDF

### Key Features
- **Local-only storage:** Evidence stays in the browser
- **Timeline builder:** Helps organize events accurately
- **AI-assisted documentation:** Converts raw inputs into a neutral, factual report
- **PDF export:** Downloads a ready-to-share document with embedded evidence

### External Resources
- jsPDF for client-side PDF creation
- Google Gemini API for AI-generated content
- No datasets are used; all input is user-provided
